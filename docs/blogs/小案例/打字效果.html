打字效果

XuJiong 2019-10-08 funny

# 效果

# 说明

  1. 通过setInterval循环，如长度小于文本字数时使文字逐个增加（可以设置打字时间）
  2. 通过::after添加__的动态打字效果
  3. 通过css3动画使文字从下到上移动（可以实现上下左右）

# 代码

  • html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <style>
      body {
        padding: 100px;
      }
      #content {
        background-color: antiquewhite;
      }
      #content::after {
        content: "__";
        animation: blink 0.2s infinite;
      }
      .span-animation {
        display: inline-block;
        animation: dance 0.2s;
      }

      @keyframes blink {
        form {
          opacity: 1;
        }
        to {
          opacity: 0;
        }
      }

      @keyframes dance {
        0% {
          opacity: 0;
          transform: translateY(300px);
        }
        100% {
          opacity: 1;
          transform: translateY(0px);
        }
      }
    </style>
  </head>
  <body>
    <div id="content"></div>
    <button onclick="sayOclick()">say</button>
    <script>
      let data = "打字效果".split("");
      function writing(text, dom) {
        let index = 0;
        let write = setInterval(() => {
          if (index < text.length) {
            dom.innerHTML =
              dom.innerHTML.replace(/span-animation/g, "") +
              `<span class="span-animation">${text[index]}</span>`;
            ++index;
          }
        }, 200);
      }
      writing(data, content);
    </script>
  </body>
</html>

  • vue组件
<template>
  <div id="write" ref="write"></div>
</template>

<script>
export default {
  name: "write",
  props: {
    text: {
      default: "Hello Word",
      type: String,
    },
  },
  data() {
    return {};
  },
  mounted() {
    this.writing(this.text, this.$refs.write);
  },
  methods: {
    writing(text, dom) {
      let index = 0;
      let write = setInterval(() => {
        if (index < text.length) {
          dom.innerHTML =
            dom.innerHTML.replace(/span-animation/g, "") +
            `<span class="span-animation">${text[index]}</span>`;
          ++index;
        }
      }, 200);
    },
  },
};
</script>

<style>
#write {
  background-color: antiquewhite;
}
#write::after {
  content: "__";
  animation: blink 0.2s infinite;
}
.span-animation {
  display: inline-block;
  animation: dance 0.2s;
}

@keyframes blink {
  form {
    opacity: 1;
  }
  to {
    opacity: 0;
  }
}

@keyframes dance {
  0% {
    opacity: 0;
    transform: translateY(300px);
  }
  100% {
    opacity: 1;
    transform: translateY(0px);
  }
}
</style>
